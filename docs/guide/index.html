# Guia

# O que é o Odin?

Odin é um projeto criado e mantido pela comunidade @wpbrasil para chutar bundas auxiliar o desenvolvimento de temas WordPress com ferramentas modernas.

# Download

  • Baixar versão 2.x .zip
  • Baixar versão 3.0.0-beta .zip

# Instalação

Envie a pasta "odin" para o diretório de temas padrão do WordPress wp-content/themes ou simplesmente envie o arquivo zipado utilizando o instalador de temas do WordPress.

# Contribuidores

O Odin é um projeto Open Source, qualquer um pode fazer parte. Para contribuir basta fazer o Fork, criar um Branch, realizar as melhorias e enviá-lo o Pull Request.
